中3追試験対策アプリ (理科マスターPro)

中学理科（化学・生物・物理）の重要語句や公式を効率よく学習・定着させるためのWebアプリです。
追試験対策や、受験前の基礎固めに最適化されています。

📖 特徴

3分野＋総合網羅

化学・生物・物理の各分野ごとに学習可能。

「全部ミックス」モードで実力テスト形式の学習も可能。

「言えた！確認」システム

単なる4択ではなく、まず「声に出して言えるか」を自己確認し、その後に選択肢で正解をチェックする2段階方式を採用。

メタ認知（自分が分かっているかを自覚すること）を促し、定着率を高めます。

習熟度管理

各問題に「5回」正解することで「習得済」と判定。

「苦手特訓」モードでは、未習得の問題だけを重点的に学習できます。

ランキング機能

学習結果をスコア化し、端末内に上位5位まで記録します。

効果音付き

正解・不正解・操作音が鳴り、リズムよく学習できます（Web Audio API使用のため外部ファイル不要）。

🎮 使い方

モード選択: 学習したい分野（化学・生物・物理・ミックス）を選びます。

問題数選択: 10問〜全問まで、隙間時間に合わせて選べます。

プレイ:

問題が表示されたら、答えを声に出してみます。

「言えた！」「言えなかった」ボタンを押します。

4択の選択肢が表示されるので、正解を選びます。

「言えた！」かつ「正解」の場合のみ、習熟度が上がります。

結果: スコアが表示され、ランキングに登録されます。

🚀 インストール・デプロイ方法

このアプリは単一の HTML ファイル (index.html) で動作します。

⚠️ ファイル作成時の注意（Macユーザーの方へ）

Macの標準アプリ「テキストエディット」などでファイルを作成すると、デフォルトでリッチテキスト（.rtf）になり、うまく動作しないことがあります。以下の手順で標準テキストとして保存してください。

テキストエディットのメニューバー「フォーマット」から**「標準テキストにする」**を選択（または Shift + Command + T）。

保存時にファイル名を README.md と入力します。

「拡張子を".md"に変更しますか？」と聞かれたら、**「.md を使用」**を選択してください。

GitHub Pages での公開手順（詳細版）

プログラミング初心者の方でも公開できるように、Webブラウザだけを使った手順を説明します。

手順1: GitHubアカウントの準備

GitHub にアクセスします。

アカウントを持っていない場合は「Sign up」から登録してください（無料です）。

手順2: 新しいリポジトリ（置き場所）を作る

GitHubにログインし、画面右上の「＋」アイコンをクリックし、「New repository」を選択します。

「Repository name」に好きな名前を入力します（例: rika-app）。

※半角英数字とハイフンがおすすめです。

「Public」（公開）が選択されていることを確認します。

「Initialize this repository with:」の項目は、特にチェックしなくても大丈夫です（チェックしてもOKです）。

画面一番下の「Create repository」ボタンをクリックします。

手順3: ファイルをアップロードする

作成したリポジトリの画面が表示されます。

「…or create a new repository on the command line」などのメッセージの下にある、「uploading an existing file」というリンクをクリックします。

画面が変わったら、手元にある index.html と、この README.md ファイルを、ブラウザの枠内にドラッグ＆ドロップします。

ファイルがリストに表示されたら、下の方にある「Commit changes」という入力欄（"Add files via upload" と書かれているところ）の下にある緑色のボタン「Commit changes」をクリックします。

手順4: 公開設定をする

リポジトリのメニューバー（Code, Issues...と並んでいるところ）から「Settings」をクリックします。

左側のサイドメニューにある「Pages」をクリックします。

「Build and deployment」セクションの「Branch」という項目を探します。

「None」と書かれているプルダウンメニューをクリックし、「main」（または master）を選択します。

隣のフォルダ選択は /(root) のままでOKです。

「Save」ボタンをクリックします。

手順5: 公開完了！

設定ページの上部に「GitHub Pages source saved.」といったメッセージが出ます。

1〜3分ほど待ちます（公開処理が行われています）。

ページを再読み込み（リロード）すると、ページ上部に「Your site is live at...」というメッセージと共に、公開URLが表示されます。

そのURLをクリックすれば、アプリが動きます！このURLを生徒さんに共有してください。
